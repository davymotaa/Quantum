# Prompt — Changelog

Use este formato para registrar qualquer mudanca no Oraculo.

```markdown
## AAAA-MM-DD — Vol X, Cap Y — Titulo da atualizacao

**Arquivo alterado:** nome_do_documento
**Tipo:** adicao / correcao / substituicao / canonizacao / remocao
**Origem:** Volume, capitulo, cena ou decisao de roteiro
**Motivo:** Por que a mudanca foi feita

**Mudanca registrada:**
- Item objetivo

**Impacto futuro:**
- Como isso deve guiar capitulos futuros
```

## Regras

- Seja especifico.
- Registre origem e motivo.
- Nao transforme changelog em resumo de capitulo inteiro.
- O arquivo principal deve ficar limpo; o historico fica aqui.
